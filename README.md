# README
## ブログアプリ
好きなテーマで記事を投稿できるアプリです。
<br>
ユーザは他ユーザや投稿に対し、お気に入りやフォローをすることができます。
<br>
スマホでの操作を想定しております。


## 作成した目的
日々感じていることを他者と共有するために作成いたしました。
<br>
話のネタを収集することや、自信の考えを整理し発信することを期待。
